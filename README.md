## WeJamz-App

## This is a social media / version control for musicians 

## Tech Stack 
- React
- PassportJS
- Express
- socket.io
- eslintConfig
- axios
- slick-carousel 
- flexbox-react 

## Setup: 

# Front-End
```sh
cd frontend
```
1. install dependencies
```sh
npm install
```
2. create a production build
```sh
npm run build
```
3. run script for client
```sh
npm start
```

# Backend

```sh
cd backend
```
1. install dependencies
```sh
npm install
```
2. run script for server
```sh
npm start
```
